# Мини‑тест по типологии Адизеса (PAEI)

12 вопросов, в каждом четыре варианта **А, Б, В, Г**. Поставьте оценку от **1** (наименее похоже) до **4** (наиболее похоже).
Соответствие вариантам: **А→P**, **Б→A**, **В→E**, **Г→I**. Максимум по каждой роли — 48.

## Локальный запуск
```bash
npm install
npm run dev
# открой http://localhost:3000
```
